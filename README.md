# AP CSP Project Showcase

Welcome to my AP Computer Science Principles (CSP) Project Showcase!

This repository contains a collection of projects I created during my AP CSP class throughout the 2023-2024 school year. All projects were built utilizing fundamental web technologies including HTML, CSS, and JavaScript.

## 🚀 Features

- **Project Gallery:** Explore various assignments and creative projects I developed over the year.
- **Interactive Apps:** Includes simple games, utilities, and data-driven applications built with JS.
- **Dark Mode Aesthetic:** The showcase features a sleek, neon-themed dark mode UI.

## 🛠️ Built With

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and animations.
- **JavaScript (Vanilla):** Dynamic interactions and application logic.

## 🏃‍♂️ Running Locally

Since this is a static site, running it locally is simple. You can use any static server or simply use the VS Code Live Server extension.

1. Clone the repository:
   ```bash
   git clone https://github.com/SkorpionG/APCSP-AppLab-Projects-2324.git
   ```
2. Navigate to the project directory:
   ```bash
   cd APCSP-AppLab-Projects-2324
   ```
3. Open `index.html` in your browser or start a local server.

## ✨ Development Tooling

To ensure the codebase stays clean and consistent, this project uses:

- **Prettier:** Code formatting for JS, CSS, HTML, and Markdown.
- **ESLint:** Linter to catch JavaScript issues.

### Useful Commands

- `npm install` - Install development dependencies.
- `npm run format` - Format all code with Prettier.
- `npm run lint` - Check JavaScript for errors.
- `npm run lint:fix` - Automatically fix JavaScript linting errors.
